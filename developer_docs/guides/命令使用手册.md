# SunPilot 命令使用手册

更新日期：2026-06-11

本文档说明当前 SunPilot 项目已实现的命令、常用环境变量、API 调试方式和开发验证命令。

## 1. 基础要求

- Node.js：`>=22.22.2 <23`
- pnpm：`>=11.5.1 <12`
- Docker / Docker Compose：本地 PostgreSQL
- daemon 默认地址：`http://127.0.0.1:3737`
- Web dev server 默认地址：`http://127.0.0.1:3738`
- 默认运行数据目录：`~/.sunpilot`

## 2. 安装、数据库和构建

```bash
pnpm install
docker compose up -d postgres
pnpm build
```

默认数据库连接：

```bash
postgresql://sunpilot:sunpilot_dev_password@localhost:5432/sunpilot
```

`pnpm install` 的 postinstall 会把 launcher 链接到 `~/.local/bin/sun`。

## 3. `sun` CLI

### 3.1 启动 daemon

```bash
sun start              # 后台启动
sun start --foreground # 前台调试
sun start --port 3738  # 指定端口
```

### 3.2 查看状态

```bash
sun status
```

### 3.3 打开 Web

```bash
sun open               # 默认 https://tradeagent.asia
SUNPILOT_WEB_URL=http://127.0.0.1:3737 sun open  # 本地
```

### 3.4 停止

```bash
sun stop
```

## 4. Web 与聊天入口

- `/`、`/chat` 进入 ChatPage。
- 通过 `/v1/ws` 使用 WebSocket JSON-RPC。
- `sun chat`、`sun ask` 尚未实现。

## 5. LLM 配置

```bash
export SUNPILOT_LLM_BASE_URL=https://api.deepseek.com
export SUNPILOT_LLM_MODEL=deepseek-v4-flash
export SUNPILOT_LLM_API_KEY=你的密钥
```

备用变量名：`DEEPSEEK_API_KEY`。

## 6. 长期保存环境变量

写入 `~/.bashrc`：

```bash
export SUNPILOT_LLM_BASE_URL=https://api.deepseek.com
export SUNPILOT_LLM_MODEL=deepseek-v4-flash
export SUNPILOT_LLM_API_KEY=你的密钥
export SUNPILOT_DATABASE_URL=postgresql://sunpilot:sunpilot_dev_password@localhost:5432/sunpilot
```

## 7. API 调试命令

```bash
# 健康检查
curl http://127.0.0.1:3737/healthz

# 就绪检查
curl http://127.0.0.1:3737/readyz

# conversation 列表
curl http://127.0.0.1:3737/v1/conversations

# 创建 conversation
curl -X POST http://127.0.0.1:3737/v1/conversations \
  -H "Content-Type: application/json" \
  -d '{"title":"测试对话"}'

# HTTP chat
curl -X POST http://127.0.0.1:3737/v1/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"用一句话介绍 SunPilot"}'

# Skills 列表
curl http://127.0.0.1:3737/v1/skills

# 重新加载 skills
curl -X POST http://127.0.0.1:3737/v1/skills/reload

# WebSocket ping
node --input-type=module -e 'const ws=new WebSocket("ws://127.0.0.1:3737/v1/ws"); ws.addEventListener("open",()=>ws.send(JSON.stringify({jsonrpc:"2.0",id:"ping_1",method:"ping",params:{}}))); ws.addEventListener("message",(event)=>{ console.log(String(event.data)); ws.close(); });'
```

## 8. 域名访问

默认公网域名：`https://tradeagent.asia`。

如需增加额外域名：

```bash
export SUNPILOT_ALLOWED_ORIGINS=https://example.com,https://www.example.com
sun stop && sun start
```

Nginx 需要保留 WebSocket upgrade：

```nginx
proxy_http_version 1.1;
proxy_set_header Upgrade $http_upgrade;
proxy_set_header Connection "upgrade";
proxy_buffering off;
```

## 9. 开发与验证命令

```bash
pnpm build                           # 全量构建
pnpm test                            # 全量测试
pnpm lint                            # 全量 lint

# 单包
pnpm --filter @sunpilot/core test
pnpm --filter @sunpilot/daemon build
pnpm --filter @sunpilot/web test

# 开发服务
pnpm dev:daemon
pnpm dev:web
```
