## 根目录

| 文件 | 职责 |
| --- | --- |
| `package.json` | monorepo 根包配置。 |
| `pnpm-workspace.yaml` | pnpm workspace 包范围声明。 |
| `tsconfig.base.json` | 全仓 TypeScript 基础编译配置。 |
| `eslint.config.js` | 全仓 ESLint flat config。 |
| `prettier.config.js` | 全仓 Prettier 格式化配置。 |
| `docker-compose.yml` | 本地 PostgreSQL 开发数据库。 |
| `scripts/link-sun-bin.mjs` | postinstall 链接 `sun`/`sunpilot` 命令。 |
